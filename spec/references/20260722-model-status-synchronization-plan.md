# Realtime Model Status Synchronization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Keep the Active Models selectors, worker detail rows, and lifecycle indicator synchronized throughout model activation and rollback.

**Architecture:** Derive one presentation snapshot from the selected target and the latest model catalog. The snapshot exposes target labels, runtime device, lifecycle, and a strict Ready predicate; `App` and `ActiveModels` consume that shared truth instead of mixing catalog state with static capabilities.

**Tech Stack:** React 18, TypeScript, Vitest, Testing Library, Vite.
